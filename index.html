Summary: Latitude Analysis for Weather

This site is a dashboard showing plots from a weather project. I used a Python library to randomly select 500+ cities based on longitude and latitude, and then the OpenWeatherMap API to pull weather data. All of the data was pulled on 28 September 2018.

The weather information was pulled with a series of successive API calls, and then the information was saved into a CSV file, which can be seen in the data section.

The project involved reviewing the dataset, and using Matplotlib to generate images of various weather aspects including temperature, humidity, cloudiness, and wind speed, plotted using latitude for cities.
